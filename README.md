# Dorm Energy Simulator

Консольное приложение на C++20 для мониторинга энергопотребления студенческого общежития.

**Цель проекта:**
Создать надёжную backend-систему, способную:
- Получать данные от IoT-датчиков 
- Обнаруживать аномалии (пороговые правила + ML-модель)
- Сохранять данные в TimescaleDB
- Отправлять уведомления в Telegram

## Текущий статус

### ✅ Завершена **Iteration 1 — CLI + Synthetic Data Generator**

В рамках первой итерации реализовано:
- Современная модульная архитектура проекта
- `SyntheticDataGenerator` — мощный генератор реалистичных данных энергопотребления с возможностью воспроизведения и добавления аномалий
- `AnomalyDetector` — детектор аномалий на основе пороговых правил
- Удобный командный интерфейс через **CLI11**
- Полная поддержка русского языка (UTF-8)
- Чистый, хорошо документированный код (Doxygen)

Это позволяет **уже сейчас** тестировать логику генерации данных и детекции аномалий без реального оборудования.

### Планируется в Iteration 2

- Режим демона (долгоживущий процесс)
- Интеграция с MQTT-брокером (приём реальных данных)
- Сохранение данных в TimescaleDB
- Система логирования
- Подготовка к подключению ONNX-модели
- Unit-тесты

## Как использовать

```bash
# Сборка проекта
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
cmake --build .

# Запуск симуляции
./dorm-sim simulate --days 30 --anomalies --verbose

# Просмотр помощи
./dorm-sim simulate --help
