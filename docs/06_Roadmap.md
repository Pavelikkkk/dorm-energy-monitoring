# Планы развития (Roadmap)

## 🗺️ Общая картина развития

**Кратко по этапам:**
- **Сейчас (0.2)** — симуляция + базовые детекторы + Telegram
- **Ближайшее (1.0)** — реальные датчики + буферизация + облако
- **Следующее (2.0)** — автоматический режим + статистика 
- **Будущее (3.0)** — ML-прогнозирование + автоотключение

## 🚀 Версия 1.0 

**Цель:** Подключение к реальному железу

### Запланированные фичи

- ESP32-драйвер                 
- Буфер данных (SQLite)        
- InfluxDB для хранения метрик 

## ⚡ Версия 2.0 

**Цель:** Постоянно работающая система с аналитикой

### Запланированные фичи

- Автоматический режим (systemd/Windows service) 
- Планировщик задач (каждую минуту/час) 
- Сбор статистики (среднее, пики, тренды) 
- Grafana для визуализации 
- Автозапуск при старте системы 

## 🌟 Версия 3.0 — Умные функции

**Цель:** Предсказание и автоматизация

### Идеи для будущего

- ML-прогнозирование и статиска - Предсказывать потребление  
- Автоотключение приборов - Реле, отключающее нагрузку при перегрузке 
- Мобильное приложение - Управление и мониторинг с телефона 
- Интеграция с умным домом - Home Assistant, OpenHAB 
- Обнаружение неисправных приборов - По паттернам потребления
